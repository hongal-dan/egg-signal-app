### How to start

🚨 node 버전 20 기준으로 개발하였습니다.

🚨 npm install 중 mind-ar canvas 오류가 생기면

```
brew install pkg-config cairo pango libpng jpeg giflib librsvg
```

위 명령어를 입력해주세요!
